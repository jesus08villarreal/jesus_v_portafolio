## 🔧 Instalación y Uso

1. **Clonar el repositorio**:
   ```bash
   git clone https://github.com/jesus08villarreal/jesus_v_portafolio.git
   cd jesus_v_portafolio
   ```

2. **Instalar dependencias**:
   ```bash
   npm install
   # o
   yarn install
   ```

3. **Ejecutar en desarrollo**:
   ```bash
   npm run dev
   # o
   yarn dev
   ```

4. **Construir para producción**:
   ```bash
   npm run build
   # o
   yarn build
   ```
